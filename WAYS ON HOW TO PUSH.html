HOW TO PUSH ON GITHUB 


    

        

            1.Open git bash
               
            2.Type the command [mkdir "YOURFILE NAME"]
            
            3.Type the command [git init]

            - to initialize empty git repository

            4.Type the command [ls]

            - to list contents of the current working directory

            5.Type the command [pwd]

            6.Type the command [echo "# Deb" >> README.md]

            7.Type the command [git init]

            8.Type the command [git add README.md]

            9.Type the command [git commit -m "first commit"] 

                  - type the command [git config --global user.email "your email"] 

                - type the command [git config --global user.name "your name"] 

            10.Type the command [git branch -M main]

            11.Type the command [git remote add origin https://github.com/"yourgithubaddress"]

            12.Type the command [git status] 

            13.Type the command [git commit -m "first commit"] 

            14.Type the command [git branch -M main] 

            15.Type the command [git remote add origin https://github.com/"yourgithubaddress"] 

            16.Type the command [git push -u origin main]
